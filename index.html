Client Intake Master Pipeline

Property Management Operations Roadmap

Internal Use Only
1

Lead Capture

Checklist

  • Capture Name, Phone, Email
  • Verify Service Area
  • Determine Lead Source

Software Actions

CRM: Auto-create Lead record

Slack: Ping #leads channel

Primary Integration

Webform ➔ Zapier ➔ HubSpot

2

Qualification

Checklist

  • Discovery Call Completed
  • Budget/Goals Discussed
  • Send "How We Work" PDF

Software Actions

Calendar: Send Calendly link

CRM: Log call notes immediately

Primary Integration

Calendly ➔ G-Cal ➔ CRM

3

Walkthrough

Checklist

  • Take Pre-Onboarding Photos
  • Create Maintenance List
  • Run Final Comp Report

Software Actions

Drive: Upload photos to shared folder

Slack: Post walkthrough summary

Primary Integration

Phone ➔ G-Drive ➔ CRM Link

4

Contracting

Checklist

  • Send Mgmt Agreement
  • Collect W-9 / Tax Info
  • Collect Keys/Codes

Software Actions

E-Sign: Send DocuSign envelope

Slack: Update #ops on status

Primary Integration

DocuSign ➔ Slack (Alert)

5

Onboarding

Checklist

  • Input into AppFolio/Buildium
  • Verify Owner Insurance
  • Schedule Lock Change

Software Actions

PM Tool: New Property Entry

Project: Create ClickUp/Monday Board

Primary Integration

CRM (Won) ➔ ClickUp Template

6

Go Live

Checklist

  • Syndicate to Zillow/MLS
  • Place Yard Signage
  • Notify Leasing Team

Software Actions

Marketing: Launch Ad Spend

Slack: CELEBRATION Notification

Primary Integration

PM Software ➔ Slack #Announce

Automation Tip: Use Zapier to link HubSpot to Slack so you never miss a new lead.
Calendar Rule: No property visits without a Calendly booking first.
Contract Rule: All DocuSign notifications must ping the #contracts channel.
